# Password Reset Simulator

A simple password reset simulator created with HTML, CSS and JavaScript.

## About the project

This project simulates a password reset flow commonly used in login and help desk environments. It was created to practice form validation, step navigation and basic JavaScript logic.

## Features

- E-mail validation
- Random verification code simulation
- Multi-step form
- Password confirmation
- Password strength indicator
- Success and error messages

## Technologies

- HTML5
- CSS3
- JavaScript

## How to run

Open the `index.html` file in your browser.

## Project purpose

This project is part of my learning path in Systems Analysis and Development, focused on front-end practice and help desk related workflows.
